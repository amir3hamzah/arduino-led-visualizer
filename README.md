# arduino-led-visualizer
Arduino Nano LED Visualizer using sound sensor

# Hardware Require
- Arduino Nano/Uno
- Sound Sensor
- Jumper Cable
- Led
- 220 Ohm Resistor

# Software to use
Arduino IDE

# Schematic 
Arduino  <----- Sound Sensor
5V              VCC
GND             GND
D7              Signal

Arduino <------ LED
D8              Anode Leg
GND             Cathode
